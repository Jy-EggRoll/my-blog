今天读了北邮的一位学长的博客，感触良多。他是 PicGo 的作者，在 22 岁时，开发了对自己人生影响重大的开源项目。这也燃起了我写一篇博客，总结自己在 2025 年的全部的心。

同时，由于互联网的特殊性，我也确实不可以和盘托出，所以还是要以最重要的事情为主，同时不暴露太多自己的个人信息。

对于隐私也是越来重视了，因为课程实验开放了端口，但是云服务器被攻击了，还对我造成了一些损失。虽然损失不大，但是对我而言是一次宝贵的学习机会。有了这次事件，我就对安全相关的事更上心了，还会主动地学习。

我也想回顾、记录自己的人生历程。

博客也已经上线一周年了。也就是去年的这个事件，我第一次开始把博客建立起来。当时也是第一次了解 hugo 这个工具，了解 stack 这个主题，刚刚开始用 git，刚刚开始用 cloudflare。如今，一年过去了，我已经产出了数十篇博客文章，开始了解了 go 语言，开始学习写 go 语言程序的最佳实践，开始维护自己的小项目，开始建立自己的交流群，开始积极地为自己的人生规划做考虑，开始将自己的诸多代码托付给 git 和 github，开始学习 ssh，远程开发……和去年的这时候比起来，自己不管是技术、眼界、心态，确实都成熟了很多。
